Network Anomaly
Detector

A data quality tool that detects and removes outlier values from SmartCare KPI CSV exports using Isolation Forest — ensuring the KPI data used for analysis and reporting accurately reflects real network conditions on XLSMART's network.

Python Isolation Forest Scikit-learn Flask CSV Data Quality SmartCare
// The Problem

KPI data exported from SmartCare contains outlier values that are physically impossible in a mobile network — latency readings of 1ms or spikes above 15,000ms that distort averages, skew standard deviations, and inflate MAX values. When these outliers are included in aggregations, KPI reports no longer reflect real network conditions, leading analysts to draw incorrect conclusions from fundamentally dirty data.

// The Solution

The tool reads SmartCare KPI CSV exports, lets the user configure which columns to include as Isolation Forest features, tunes contamination and estimator parameters, then outputs two files: a CSV with anomaly flags for audit, and a cleaned CSV with outliers removed — ready for analysis, aggregation, and loading into SQL Server via CSV Watcher.

// features
Key Features
CSV Drag & Drop
Upload any SmartCare KPI export via drag & drop. Columns are auto-detected with null counts, unique values, and type suggestions — adjustable before running the model.
Column Type Control
Toggle columns in or out of the model. Numeric columns become Isolation Forest features; categorical columns become dimension filters for post-detection slicing.
Tunable IF Parameters
Configure contamination rate, number of estimators, max samples, and random state directly from the UI — no code changes needed to tune the model per dataset.
Statistical Evaluation
Side-by-side comparison of Mean, Std Dev, Median, Min, and Max before and after dropping anomalies — with percentage change shown for every KPI column.
Interactive Scatter Plot
Visualize normal vs anomaly points on a configurable scatter plot. X and Y axes are selectable per KPI column — making outlier patterns immediately visible.
Two Output Formats
Download CSV with anomaly flags for full audit trail, or download the cleaned CSV with outliers removed — ready to load into SQL Server via CSV Watcher.
